# Time Series & Category Charts

## Project Overview
This project analyzes sales data using Python and visualizes it using different chart types.

## Features
- Monthly Sales Line Chart
- Quarterly Sales Line Chart
- Category-wise Bar Chart
- Category Share Pie Chart
- Sales Summary Export

## Technologies Used
- Python
- Pandas
- Matplotlib

- charts.py
- sales_data.csv

- category_sales.png
